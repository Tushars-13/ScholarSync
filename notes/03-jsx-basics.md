# Day 3 - JSX Basics

## What is JSX?

JSX (JavaScript XML) is a syntax that allows us to write HTML-like code inside JavaScript.

Example:

<h1>Hello</h1>

JSX looks like HTML but it is not HTML.

---

## What is <div>?

<div> is a container used to group multiple HTML elements.

It is not visible in the browser.

---

## Why do we use <div>?

A React component must return a single parent element.

<div> is commonly used as that parent container.

---

## Curly Braces {}

Curly braces {} are used to write JavaScript expressions inside JSX.

Examples:

const name = "Tushar";

<p>Welcome {name}</p>

<p>{10 + 20}</p>

<p>{name.toUpperCase()}</p>

---

## Key Points

✓ JSX looks like HTML but is not HTML.
✓ Browser cannot understand JSX directly.
✓ React components return one parent element.
✓ <div> groups multiple elements.
✓ {} allows JavaScript inside JSX.
